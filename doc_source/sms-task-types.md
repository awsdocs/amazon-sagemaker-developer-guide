# Built\-in Task Types<a name="sms-task-types"></a>

Amazon SageMaker Ground Truth has several built\-in task types\. Ground Truth provides a worker task template for buit\-in task types\. Additionally, some built in task types support [Automate Data Labeling](sms-automated-labeling.md)\. The following topics describe each built\-in task type and demo the worker task templates that are provided by Ground Truth in the console\. To learn how to create a labeling job in the console using one of these task types, select the task type page\.
